Проблема

Заказчик обратился к нам с просьбой создать приложение, с помощью которого можно было бы дистанционно управлять работой видеорегистратора с мобильного устройства - телефона или планшета.

Решение

Мы разработали многофункциональное Android-приложение c двуязычным интерфейсом. Взаимодействие между приложением и видеорегистратором осуществляется по REST API.
Основными функциями приложения являются:

  • соединение с регистратором по wifi / gsm;
  • настройка запуска работы видеорегистратора по акустическому сигналу, определению движения;
  • настройка расписания для запуска видео- и аудиозаписи;
  • управление трансляцией потокового видео;
  • управление характеристиками записи видео;
  • обеспечение экспорта/импорта видео- и аудиозаписей;
  • хранение/удаление видеозаписей;
  • определение геолокации;
