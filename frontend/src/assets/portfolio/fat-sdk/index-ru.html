Проблема

В целях оптимизации рабочих процессов заказчику потребовалось внедрить систему автоматической генерации лицензионных ключей для своего продукта.

Решение

С помощью Spring Boot мы в короткие сроки разработали сервер лицензий, отвечающий за создание, преобразование, шифрование ключей, и создали графический интерфейс для управления ключами, с помощью которого менеджеры легко и удобно могли создавать и удалять ключи, отслеживать количество созданных ключей, период их действия, статус и прочие характеристики.
