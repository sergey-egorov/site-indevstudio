Проблема
Заказчик, занимающийся продажей мобильных устройств, решил внедрить для своих клиентов программу trade-in. Для того, чтобы ускорить оценку стоимости устройства и сделать ее более объективной и прозрачной для клиентов, появилась необходимость в специализированном приложении.

Решение
Мы разработали решение, включающее в себя следующие компоненты:

  • Мобильное приложение для Android/iOS, которое проводит диагностику программных и аппаратных функций и характеристик мобильного устройства по нескольким десяткам параметров и передает результаты оценки в центральную систему.

  • Веб-портал, состоящий из двух модулей: - front-end-приложения, созданного на платформе Angular, - back-end-приложения, которое представляет собой standalone-приложение с базой данных, созданное на Java с использованием библиотеки Spring Boot.
    С помощью портала происходит регистрация параметров сессий диагностики мобильных устройств и автоматическое формирование отчетов и прочих документов по результатам диагностики.

  • Серверное приложение, осуществляющее централизованное управление распределенными точками диагностики мобильных устройств.
