Проблема

Потребность заказчика заключалась в необходимости наладить процесс контроля рабочего времени сотрудников, а именно фиксации времени появления сотрудника на рабочем месте и момента ухода сотрудника с работы, без использования традиционных способов в виде турникетов, пропусков и охранников. Желанием заказчика было использовать лицевую биометрию для идентификации сотрудников.

Нашей задачей было создание приложения, способного распознавать лицо и записывать данные о времени прихода/ухода сотрудника. Кроме того, приложение должно уметь сверять время появления сотрудника с установленным расписанием и на основе этого принимать решение о допуске/недопуске сотрудника к рабочему месту.

Решениe

Для решения поставленной задачи мы создали несколько компонентов:

  • Android-приложение для планшета, который располагается на входе и фиксирует сотрудника в начале и конце рабочего дня. Для реализации лицевой биометрии мы использовали технологию нашего партнера, компании ЦРТ.

  • Сервер, созданный с помощью Spring Boot, с базой данных для хранения информации о сотрудниках, который, взаимодействуя с приложением, сличает биометрические данные, создает и хранит записи о приходе/уходе сотрудников.

  • Десктопное приложение на Angular’е для администратора, которое позволяет настроить и управлять списком сотрудников, их данными, обеспечить синхронизацию с мобильным приложением, просмотреть записи о приходе/уходе сотрудников.
