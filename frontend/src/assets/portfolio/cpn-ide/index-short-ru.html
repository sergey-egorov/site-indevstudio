Приложение для построения моделей динамических систем и анализа их свойств с использованием цветных сетей Петри, графически ориентированного языка моделирования. Типичными областями применения являются сети передачи данных, паралелльные вычисления, бизнес-процессы, производственные системы и мультиагентные системы.
